Overview:

This project is a backend API built using Django and Django REST Framework to manage financial transactions. It allows users to create, view, update, and delete transaction records with proper structure and basic access control

Tech Stack:

Python
Django
Django REST Framework
SQLite (default database)

Features:

Add new transactions
View all transactions
Update transaction details
Delete transactions
Filter transactions

Install dependencies:

pip install -r requirements.txt

Run migrations:

python manage.py makemigrations
python manage.py migrate

Start the server:

python manage.py runserver
